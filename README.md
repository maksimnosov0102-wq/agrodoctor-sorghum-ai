# AgroDoctor Sorghum AI

## Місія

Створення першої спеціалізованої AI-платформи для селекції, насінництва, технологій вирощування та біоенергетичного використання сорго.

## Основні модулі

### SorghumGPT

AI-консультант з сорго.

### Breeding Database

База селекційних матеріалів, ліній та гібридів.

### Field Trials

Польові випробування.

### Seed Production

Насінництво.

### Plant Protection

Хвороби, шкідники, гербіциди.

### Bioenergy

Біомаса, біоетанол, тверде паливо.

### Analytics

Статистика та прогнозування.
